# 情感分析项目（BERT）

这是一个可落地的中文情感分析项目模板，覆盖 **项目背景、关键技术亮点、实现意义、效果展示与代码实现**。

## 1. 项目背景

在电商评论、社交媒体、客服工单等场景中，文本情感分析能够帮助企业快速感知用户满意度，及时发现负面舆情并辅助产品优化。本项目以中文文本为对象，基于预训练 BERT 构建二分类（正向/负向）情感识别系统。

## 2. 亮点关键技术

1. **预训练模型迁移学习（BERT）**  
   通过 `bert-base-chinese` 微调，利用大规模语义知识提升小样本任务效果。
2. **标准化训练评估流程**  
   包含训练集/验证集拆分、`Accuracy + F1` 指标评估，保证模型可比较、可复现。
3. **服务化部署能力（FastAPI）**  
   将模型封装为 HTTP 接口，便于前端、业务系统或自动化流程调用。
4. **工程化结构**  
   模块拆分为数据处理、模型构建、训练、评估、预测与 API 层，便于后续扩展。

## 3. 实现意义

- **业务价值**：自动识别差评，减少人工阅读成本，提升运营响应效率。  
- **技术价值**：提供从离线训练到在线推理的完整样例，可直接迁移到真实业务数据。  
- **教学价值**：代码简洁清晰，适合课程设计、毕业项目、简历项目演示。

## 4. 项目结构

```text
sentiment-analysis-bert/
├── app.py                # FastAPI 推理服务
├── data/train.csv        # 示例训练数据
├── requirements.txt
├── run.sh
└── src/
    ├── dataset.py        # 数据读取与 DataLoader
    ├── evaluate.py       # 验证指标计算
    ├── model.py          # 模型与 tokenizer 构建
    ├── predict.py        # 单条文本预测
    └── train.py          # 训练入口
```

## 5. 快速开始

### 5.1 安装依赖

```bash
pip install -r requirements.txt
```

### 5.2 训练模型

```bash
python -m src.train --epochs 3 --batch-size 8
```

训练完成后模型将保存到 `artifacts/model`。

### 5.3 命令行预测

```bash
python -m src.predict --text "物流非常快，产品质量也很好"
```

### 5.4 启动 API 服务

```bash
uvicorn app:app --host 0.0.0.0 --port 8000
```

请求示例：

```bash
curl -X POST http://127.0.0.1:8000/predict \
  -H "Content-Type: application/json" \
  -d '{"text":"客服态度很差，不会再买"}'
```

## 6. 预期实现效果

- 在示例数据上可完成正负向情感分类流程闭环。  
- 基于预训练模型，在真实数据下通常能获得比传统机器学习更高的语义理解能力。  
- 可进一步扩展为：多分类情感、方面级情感分析、舆情预警系统。
